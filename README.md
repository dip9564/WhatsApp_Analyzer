# 💬 WhatsApp Chat Analyzer

An AI-powered WhatsApp Chat Analyzer built with **Python**, **Streamlit**, **Scikit-learn**, and **Google Gemini AI**. Upload your exported WhatsApp chat and explore detailed analytics, visualizations, sentiment analysis, interaction networks, and AI-generated conversation summaries.

## 🌐 Live Demo

👉 **Live App:** https://advance-chat-analyzer.streamlit.app

## ✨ Features

### 📊 Chat Analytics
- Total Messages, Words, Media Shared, Links Shared

### 📈 Activity Analysis
- Monthly Timeline
- Daily Activity
- Weekly Activity
- Weekly Heatmap

### ☁️ Content Analysis
- Word Cloud of Most Frequent Words
- Emoji Analysis

### 🤝 Conversation Insights
- Response Time Analysis
- Night Owl Analysis
- Conversation Starters
- Interactive Conversation Network

### 😊 Sentiment Analysis
- Positive & Negative Sentiment (Participant-wise)
- Most Positive & Negative Messages

### ⚖️ Member Comparison
Compare any two participants based on:
- Messages
- Words
- Response Time
- Sentiment
- Monthly Activity
- Conversation Behaviour

### 🤖 AI Features
- AI Chat Summary (Google Gemini)
- Main Topics
- Important Decisions
- Action Items
- Overall Conversation Mood

## 📂 How to Use

1. Export a WhatsApp chat (**Without Media** recommended).
2. Open the app.
3. Upload the exported `.txt` or '.zip' file.
4. Select a participant or choose **Overall**.
5. Explore the analytics and AI insights.

---

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/dip9564/WhatsApp_Analyzer.git
```

Go to the project directory:

```bash
cd whatsapp-chat-analyzer
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
